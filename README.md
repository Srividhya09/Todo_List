# Todo_List
To-Do List Application
This is a simple To-Do List application built using Tkinter, a standard GUI library for Python. The application allows users to manage tasks by adding, deleting, marking as complete, and updating tasks. Tasks are stored in a text file and loaded when the application starts.

# Features
Add Task: Add a new task with a description, priority, and due date.
Delete Task: Delete an existing task from the list.
Mark Complete: Mark a task as completed.
Update Task: Update the details of an existing task.
Filter Tasks: Filter tasks to show only pending or completed tasks.
Show All Tasks: Display all tasks regardless of their status.

# Requirements
Python 3.12
Tkinter (comes pre-installed with Python)
datetime module (comes pre-installed with Python)

# Installation
Clone the repository:
git clone https://github.com/Srividhya09/Todo_List.git

Navigate to the project directory:
cd Todo_List
Ensure you have Python 3.12 installed. You can download it from here.

# Usage
Navigate to the project directory:
cd Todo_List
Run the application:
python todo_list.py
The main window will open, displaying the list of tasks. You can use the buttons to add, delete, mark as complete, update, filter, or show all tasks.

# Code Overview
Task Class: A class to represent a task with attributes for description, priority, due date, and completion status.
load_tasks: Function to load tasks from a text file.
save_tasks: Function to save tasks to a text file.
ToDoListApp Class: The main application class that sets up the GUI and handles user interactions.
